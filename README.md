# Flipkart Mobile Dataset Analysis (EDA using Python)

## Description
This project performs Exploratory Data Analysis (EDA) on a Flipkart mobile dataset to uncover insights about smartphone features, pricing, discounts, and sales performance. Using Python libraries, various visualizations are created to analyze patterns and relationships between different variables.

## Objectives
Analyze brand-wise performance
Understand price distribution and discount impact
Explore relationship between sales and features (RAM, ROM, battery, etc.)
Identify key factors influencing customer ratings and sales

## Key Analysis & Visualizations
📌 Countplot → Brand-wise product distribution
📌 Bar Charts → Average & Total Sales by Brand
📌 Donut Chart → Brand-wise Units Sold
📌 Histogram → Price & Battery Capacity Distribution
📌 Boxplot → Feature comparison (RAM, Battery, Price)
📌 Scatter Plot → Sales vs Discount, Price vs Rating
📌 Correlation Heatmap → Feature relationships

## Key Insights
-Certain brands dominate in both product count and total sales
-Higher discounts generally lead to increased sales
-Features like RAM, battery capacity, and display size show strong relationships
-Number of ratings highly influences sales performance
